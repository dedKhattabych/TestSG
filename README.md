 # Краткая Инструция
Как запускать (что доустановить, как скомпилировать, как настроить).

Чтобы импортировать .tsv файл в базу данных необходимо перетащить его в .exe
Строка подключения к бд в .NET 5 содержится в appsettings.json.
Используется postgreSQL 14 https://www.postgresql.org/download/

Строка подключения к бд "Host=localhost;Port=5432;Database=mydatabase;Username=postgres;Password=1"

Чтобы вывести текущую структуру данных необходимо запустить .exe файл и ввести в консоль ID подразделения.
Если необходимо вывести все подразделения (иерархия учитывается) необходимо ввести 0.
